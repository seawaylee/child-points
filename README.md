# 积分小管家

微信小程序 - 孩子积分管理系统。家长通过设定任务让孩子赚取/消费积分（1积分 = 1分钟娱乐时间），培养好习惯。

## 功能特性

- **积分管理** - 赚积分 / 花积分，实时余额和今日统计
- **灵活任务系统** - 自定义任务、时长类/次数类计分、积分倍率、启用/禁用
- **拍照记录** - 每次记录支持拍照上传作为凭证
- **历史流水** - 按日分组、周统计、分页加载、长按撤销回滚
- **多人协作** - 邀请码邀请家人，管理员可管理成员角色、移除成员
- **月度赠送** - 每月自动赠送积分，额度可配置
- **数据隔离** - 每个家庭数据完全隔离，互不可见

## 技术栈

- 微信小程序（WXML / WXSS / JS）
- 微信云开发（云函数 + 云数据库 + 云存储）
- 无第三方 UI 框架

## 项目结构

```
cloudfunctions/          # 云函数
├── login/               # 用户登录
├── family/              # 家庭创建/加入/管理/角色
├── task/                # 任务增删改查
├── record/              # 积分记录与撤销
└── stats/               # 统计数据
miniprogram/
├── pages/
│   ├── index/           # 首页 - 积分展示 + 任务快捷入口
│   ├── setup/           # 初始设置
│   ├── record/          # 记录积分
│   ├── record-detail/   # 记录详情
│   ├── history/         # 积分流水（支持长按撤销）
│   ├── family/          # 家庭管理（成员角色、邀请码、月度赠送）
│   └── task-manage/     # 任务管理（仅管理员）
└── utils/
    ├── cloud.js         # 云函数调用封装
    └── util.js          # 工具函数
```

## 快速开始

1. 安装[微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 微信扫码登录
3. Clone 项目并导入开发者工具
4. 右键各云函数目录 → 上传并部署：云端安装依赖

详细说明见 [docs/部署指南.md](docs/部署指南.md)，完整功能介绍见 [docs/功能介绍.md](docs/功能介绍.md)。
