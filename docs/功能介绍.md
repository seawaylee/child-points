# 积分小管家 - 功能介绍

## 产品概述

积分小管家是一款微信小程序，帮助家长通过积分机制培养孩子的好习惯。孩子通过完成积极活动（看书、做作业、运动等）赚取积分，进行消费活动（看电视、玩游戏等）时消耗积分。**1 积分 = 1 分钟**，让孩子直观理解时间价值。

---

## 核心功能

### 1. 积分管理

- **赚积分**：孩子完成指定活动（看书、做作业、练字、运动、做家务等）获得积分
- **花积分**：孩子进行消费活动（看电视、玩游戏、吃零食等）消耗积分
- **实时余额**：首页大字展示当前可用积分，直观清晰
- **今日统计**：实时显示当日获得和消费的积分数

### 2. 灵活的任务系统

- **自定义任务**：家长可自由添加、编辑、删除任务类型
- **双向任务**：支持「赚积分」和「花积分」两种任务类型
- **计分方式**：支持「时长类」（按分钟计时）和「次数类」（按次计分）两种模式
- **积分倍率**：每个任务可设置不同的积分数（默认 1 分/分钟 或 1 分/次）
- **自定义图标**：每个任务可设置 emoji 图标，在首页直观展示
- **预设模板**：创建家庭时自动生成 8 个常用任务，开箱即用
- **启用/禁用**：可临时禁用不需要的任务，无需删除

### 3. 拍照记录

- **拍照上传**：每次记录积分时支持拍照或从相册选择照片
- **凭证留存**：照片作为活动凭证保存在云端，流水记录中可查看
- **大图预览**：在历史流水中点击缩略图可查看原图

### 4. 历史流水

- **完整记录**：保留所有积分变动记录
- **按日分组**：自动按日期分组展示，标注"今天"、"昨天"
- **详细信息**：每条记录包含任务名、积分变动、时长、操作人、照片、备注
- **周统计**：顶部显示本周获得和消费的积分汇总
- **分页加载**：支持下拉刷新和触底加载更多
- **撤销回滚**：长按记录可撤销该条记录并自动回滚积分

### 5. 多人协作

- **家庭共享**：通过 6 位邀请码邀请其他家长加入家庭
- **共同管理**：所有家庭成员均可为孩子记录积分
- **角色区分**：创建者为管理员（可管理家庭和任务），加入者为普通成员
- **角色管理**：管理员可将普通成员提升为管理员，也可移除成员
- **操作留痕**：每条记录标注操作人，知道是谁给孩子记的分

### 5.5 月度赠送

- **每月自动赠送**：每月首次打开小程序时自动赠送设定额度的积分
- **额度可配**：管理员可在家庭管理中调整每月赠送积分数量（默认 100 积分）
- **自动判断**：系统自动判断当月是否已赠送，避免重复

### 6. 权限控制

系统分为两种角色，权限如下：

| 功能 | 管理员（创建者） | 普通成员（加入者） |
|------|:---:|:---:|
| 记录积分 | ✅ | ✅ |
| 查看积分余额 | ✅ | ✅ |
| 查看历史流水 | ✅ | ✅ |
| 查看任务列表 | ✅ | ✅ |
| 撤销自己的记录 | ✅ | ✅ |
| 撤销他人的记录 | ✅ | ❌ |
| 修改家庭信息 | ✅ | ❌ |
| 添加/编辑/删除任务 | ✅ | ❌ |
| 管理成员角色/移除成员 | ✅ | ❌ |
| 查看邀请码 | ✅ | ✅ |

### 6. 快捷操作

- **首页快捷按钮**：直接在首页点击任务即可快速记录
- **预设时长**：提供 5/10/15/30/60 分钟快捷选项，也支持自定义输入
- **一键复制邀请码**：方便通过微信分享给家人

## 页面说明

### Tab 1：积分（首页）

展示孩子当前可用积分和今日统计。下方分为两组：
- **赚积分**：绿色卡片，展示所有赚取类型的任务按钮
- **花积分**：红粉色卡片，展示所有消费类型的任务按钮

点击任意任务按钮 → 跳转到记录详情页完成积分记录。

右上角设置按钮进入家庭管理。

### Tab 2：记一笔

展示所有任务的完整列表，按「赚积分」和「花积分」分组。每个任务显示图标、名称、类型标签和积分倍率。点击进入记录详情页。

### Tab 3：流水

展示积分变动的历史记录。顶部为本周统计卡片。下方按日期分组，每条记录显示任务名、积分变动（+绿色 / -红色）、操作人、时间和照片缩略图。

### 记录详情页（非 Tab）

从首页或记一笔页面点击任务进入。包含：
- **时长选择**：预设 5/10/15/30/60 分钟按钮 + 自定义输入
- **拍照**：点击拍照按钮选择或拍摄照片，支持预览和删除
- **备注**：可选的文字备注输入
- **提交**：一键提交积分记录

### 家庭管理页（非 Tab）

三种模式：
- **创建家庭**：输入家庭名称和孩子姓名，创建后自动成为管理员
- **加入家庭**：输入 6 位邀请码加入已有家庭
- **家庭详情**：查看和编辑家庭信息、邀请码、成员列表、进入任务管理

### 任务管理页（非 Tab）

管理所有任务模板：
- 按赚/花分组展示，每个任务显示图标、名称、积分倍率、启用状态
- 底部「添加任务」按钮
- 点击任务可编辑
- 左滑可删除
- 支持启用/禁用切换
